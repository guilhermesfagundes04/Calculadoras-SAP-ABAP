# Calculadoras-SAP-ABAP
Exercício 1: - Montar uma calculadora com as 4 operações para 2 números.    
Exercício 2: - Montar uma calculadora com as 4 operações para 4 números (algum número informado for 0 ou branco) não considerar.    
Exercício 3: - Ler uma data em formato DD/MM/AAAA e imprimir no formato DD de MMMM de AAAA. Exemplo: 15/01/2016 -> 15 de Janeiro de 2016.
