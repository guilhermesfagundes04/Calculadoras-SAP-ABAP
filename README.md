# Calculadoras-SAP-ABAP
Ex. 1:    - Montar uma calculadora com as 4 operações para 2 números.    Ex. 2:    - Montar uma calculadora com as 4 operações para 4 números (algum número informado for 0 ou branco) não considerar.    Ex. 3:    - Ler uma data em formato DD/MM/AAAA e imprimir no formato DD de MMMM de AAAA. Exemplo: 15/01/2016 -> 15 de Janeiro de 2016.
